# Tailwind Project

This is a simple project built with [Tailwind CSS](https://tailwindcss.com/).

## Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
npm install
```
